# e-Commerce Example

This is a small project to implement an API in C# which will manage Customers and Orders.

Key Features
* Customers actions.
* Products actions. 
* Orders actions.
* Customers' orders actions.



## How to run the project

* Change DB connection string which can be located in `ECommerce.Example/API/appsettings.json` => ConnectÄ±onStrings section
 
* Run API project and you can see swagger API UI to test API.
